## Vagrant commands

####  Creates Vagrant environment by creating an initial Vagrantfile

` vagrant init hashicorp/precise64`

#### Validate your Vagrantfile

`vagrant validate`

#### Start machine

`vagrant up`

#### Stop machine

`vagrant halt`

#### Connect as root

`vagrant ssh`

#### Check if your machine is running

`vagrant status`

#### Delete a machine

`vagrant destroy`

#### Check your vagrant version

`vagrant --version`
